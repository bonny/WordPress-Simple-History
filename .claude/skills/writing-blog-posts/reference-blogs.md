# Reference Blogs

These blogs inform the voice and style for Simple History content. Consult them for additional inspiration.

## General Product Updates

| Blog | Style | Use For |
|------|-------|---------|
| [Panic Blog](https://blog.panic.com/) | Friendly, human, honest | Product announcements, difficult news |
| [Search & Filter](https://searchandfilter.com/blog/) | Enthusiastic, metric-driven | Version updates, feature roundups |
| [date-fns](https://blog.date-fns.org/) | Celebratory but technical | Release notes, breaking changes |
| [Josh Comeau](https://www.joshwcomeau.com/blog/) | Personal, story-driven | Technical tutorials, behind-the-scenes |
| [Claude Blog](https://claude.com/blog/) | Confident, accessible | Feature explanations, guides |

## Technical Documentation

| Blog | Style | Use For |
|------|-------|---------|
| [WordPress Developer News](https://developer.wordpress.org/news/) | Empathetic expert | Tutorials, best practices |
| [Make WordPress Core](https://make.wordpress.org/core/) | Community-focused | Release notes, technical decisions |
| [MDN Web Docs](https://developer.mozilla.org/) | Clear, comprehensive | API documentation, technical reference |
